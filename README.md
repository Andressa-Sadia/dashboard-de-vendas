# Dashboard de Vendas

Projeto front-end simples para portfólio e GitHub, com foco em organização e visualização de dados de vendas.

## Estrutura
- `index.html` → layout principal
- `styles.css` → visual do dashboard
- `script.js` → dados mockados, filtros, KPIs e gráficos

## Funcionalidades
- KPIs principais
- Gráfico de faturamento por mês
- Comparativo por região
- Ranking por vendedor
- Participação por categoria
- Filtros por mês, região e vendedor
- Tabela com vendas filtradas

## Como usar
1. Baixe a pasta ou clone o repositório.
2. Abra o arquivo `index.html` no navegador.
3. Edite o array `salesData` em `script.js` para usar seus dados.
